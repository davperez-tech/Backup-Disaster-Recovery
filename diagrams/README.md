# Diagrams

Store network architecture diagrams and backup flow diagrams here.

**Recommended tools for creating diagrams:**
- [draw.io](https://app.diagrams.net/) — free, exports to .drawio (editable) and .png
- [Excalidraw](https://excalidraw.com/) — free, hand-drawn aesthetic
- Microsoft Visio — if available

**Suggested diagrams to include:**
- Network architecture (VLANs, IPs, routing through pfSense)
- Backup data flow (primary backup → local repo → S3 copy job)
- Restore decision tree (file-level vs bare-metal vs ransomware recovery)
